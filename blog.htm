A full version of Gridloch is underway!

slurpinpuffs -

I'm happy to announce that I've been working on a full version of Gridloch, my recent 72-hour game jam game! 

The game jam version had 4 levels, and the full version is planned to have a total of at least 16, with an additional 16 fast mode levels. The project is currently still a work in progress, BUT a playable version is available right now!

So, what's new so far?:

  • Added a visible target so you can see where enemies are going next
  • Added checkpoints buoys!
  • Total revamp of level 4
  • Completely new track for the Ocean levels
  • A new enemy type -- sharks! 

What else is planned?:

  • 12 completely new levels
  • 3 new areas (including the new Forest River area in the above screenshot!), each with their own unique theme and music track
  • The game gets progressively faster as levels continue
  • Tons of new enemies with unique patterns and movements
  • Nessy going to spaaaace!?

Stay tuned for more, y'all! <3
